🖥️

Frontend — Next.js

App Router with role-based navigation. 20+ pages across 9 sections. Shared UI library with shadcn-style components (Button, DataTable, FormField, workflow primitives). Tailwind CSS styling.

⚡

API — Lambda Handlers

Bounded-context service architecture. Lambda handlers behind API Gateway. Middleware stack: JWT auth → permission guard → scope guard → row-level access → audit context → error handling.

🗄️

Database — PostgreSQL

Aurora Serverless v2 (Docker locally). Prisma ORM with 13 schemas, 40+ models. Repository pattern for complex transactional writes. 4 incremental migrations.

📡

Events — Outbox + Workers

80+ domain event types. Transactional outbox pattern ensures at-least-once delivery. Worker runtime with retry logic, subscriber registry. EventBridge in production.

Monorepo Packages

9 internal packages + 3 apps providing shared infrastructure across the platform.

📐

packages/domain

Canonical domain contracts — entity types, event catalog (80+ events), audit interfaces, observability metrics, VIN generator. The shared language of the system.

🗄️

packages/db

Prisma schema (1300+ lines, 40+ models, 13 schemas), database client, 9 repository modules. The persistence layer source of truth.

🔐

packages/auth

JWT handling, password utilities, RBAC engine, 17 permissions, 8 roles, scope management, row-level access, session primitives.

📡

packages/events

Event bus abstraction, typed event contracts, publisher/subscriber interfaces. Foundation for the event-driven architecture.

📅

packages/scheduling

Deterministic build-slot planning engine. Greedy algorithm with priority/due-date sorting for shop-floor capacity planning.

🤖

packages/ai

AI provider port interface, services (work order summarization), Bedrock integration scaffolding. Pluggable AI capabilities.

🔄

packages/migration

Full ShopMonkey migration pipeline — CLI tools (fetch, sanitize), API connector, CSV parsers, transformers, validators, dedup engine, staged loaders.

🎨

packages/ui

Reusable UI primitives (Button, DataTable, FormField), theme tokens, workflow components (BlockedReasonDialog, QueueList, SyncStatusBadge).

🧪

packages/test-utils

Cross-project architecture and contract verification tests. Ensures structural integrity across packages.
